Installation

Pour facilement essayer ReasonReact, nous offrons deux solutions ayant différents objectifs.

Bsb #

Objectifs: simplicité, contrôle, application traditionnelle avec plusieurs fichiers html.

Prérequis: avoir installé bsb, via npm install -g bs-platform. D'avantage d'instructions concernant l'installation ici.

bsb -init my-react-app -theme react

Le système de build de BuckleScript bsb a une commande init qui génère un modèle de projet. Le thème react offre une solution légère et optimisée pour un apprentissage sans complexifications et pour une facilité d'intégration dans un projet existant.

It compiles to straighfoward JS files, so you can open index.html directly from the file system. No server needed.

Reason Scripts (avec serveur de développement) #

Objectifs: single-page application (SPA), s’appuie sur les bonnes pratiques provenant de la communauté create-react-app.

Reason-scripts fournit une expérience familière aux utilisateurs ReactJS qui sont déjà habitués à create-react-app. C’est une solution globale. Cependant, si cette solution est trop lourde à votre goût, essayez la première option ci-dessus (bsb).

Comme pour create-react-app, reason-scripts est livré avec un serveur et intègre le hotloading.
