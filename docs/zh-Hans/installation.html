安装

为了简化使用ReasonReact的流程，我们提供了两种目的不同的途径。

Bsb #

目标：简单，可掌控，更传统的直接生成一些HTML文件

先决条件： 通过 npm install -g bs-platform 安装bsb。 进一步安装说明 在这里。

<code>bsb -init my-react-app -theme react</code>

BuckleScript的 bsb 构建脚本有一个 init命令，可以生成一个项目模板。 react模板提供了一个轻量级的解决方案，学习成本低而且可以很好的和现有的项目融合。

Reason代码会直接编译成JS文件，所以你可以直接打开 index.html ，而不用运行任何服务。

Reason Scripts (with Development Server) #

目标: 单页面应用(SPA)，使用create-react-app生态里的最佳开发实践

Reason-scripts 提供了一个类似于create-react-app的开发体验，让熟悉ReactJS的同学可以很轻松的上手。 它是一个全方位的解决方案。 但是如果你觉得它包含太多东西，可以尝试以上 (bsb) 的第一个选项。

与create-react-app相似，reason-scripts 自带了开发服务器和集成的hot loading。
