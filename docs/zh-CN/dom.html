ReactDOM

ReasonReact 的 ReactDOM 模块称为 ReactDOMRe， 该模块提供了类似 ReactJS 语法的一些辅助方法。 例如，你可以使用 (ReactDOMRe.domElementToObj (ReactEventRe.Form.target event)) ##value 危险地访问 event.target.value。

  • render : ReasonReact.reactElement => Dom.element => unit

  • unmountComponentAtNode : Dom.element => unit

  • findDOMNode : ReasonReact.reactRef => Dom.element

  • objToDOMProps : Js.t {..} => reactDOMProps (参考 Invalid Prop Name 中的例子)

  • domElementToObj : Dom.element => Js.t {..}: 把一个 DOM 元素转换成一个 JS object 使你可以危险的使用其中的字段。

还有另外两个便利的方法：

  • renderToElementWithClassName : ReasonReact.reactElement => string => unit: 找到第一个符合 class 名字的元素并 render 到它里面。

  • renderToElementWithId : ReasonReact.reactElement => string => unit: 找到对应 ID 的元素并 render 到它里面。

ReactDOMServer

ReasonReact's 中类似的是 ReactDOMServerRe 模块：

  • renderToString : ReactRe.reactElement => string

  • renderToStaticMarkup : ReactRe.reactElement => string
